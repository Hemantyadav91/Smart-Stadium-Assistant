# Smart Stadium Assistant

## 🚀 Overview
Smart Stadium Assistant enhances the live event experience at large-scale sporting venues by providing real-time crowd insights, intelligent route suggestions, and dynamic alerts to help attendees navigate efficiently and avoid congestion.

---

## 🎯 Problem
Large stadiums often suffer from:
- Crowd congestion
- Long waiting times
- Poor navigation experience

---

## 💡 Solution
This project simulates real-time crowd conditions and provides:
- Smart navigation suggestions
- Live congestion alerts
- Adaptive UI updates based on crowd changes

---

## ✨ Features
- 🔄 Real-time crowd simulation
- 🤖 Smart route recommendations
- 🚨 Dynamic alert system
- ♿ Accessibility-friendly UI
- ⚡ Fast and responsive interface

---

## 🛠️ Tech Stack
- React (Vite)
- JavaScript
- Vanilla CSS
- Google Cloud Run (Deployment)

---

## 🧠 How It Works
- Crowd levels are simulated dynamically
- Each zone updates in real-time
- Smart Assistant analyzes data to suggest optimal routes
- Alerts notify users about high congestion areas
